# **أداة OSD**

**ملف OSD** من الملفات التي تحتاج أداة مخصصة للتعديل، وبما أن عدد الأشخاص الذين يعرفون طريقة تعديله قليل، تم إنشاء **أداة سهلة الاستخدام** لتعليم كيفية تعديل هذا الملف بطريقة واضحة ومنظمة.

---

## **مميزات الأداة**

1. **سهولة الاستخدام**  
   تتوفر **3 ملفات BAT** لكل خيار تريده، لتسهيل عملية التشغيل.

2. **نظام تجميع ذكي**  
   الأداة تقوم بتجميع الملفات بطريقة **مرتبة وسلسة**.

3. **إضافة الأكواد مباشرة من الأداة**  
   يمكنك إدخال الأكواد اللازمة لتشغيل **OSD** دون الحاجة لتعديل الملف يدويًا باستخدام برنامج **HXD**.

---

## **طريقة التشغيل**

يمكنك تشغيل الأداة بعدة طرق:

- تشغيل ملف **BAT** الخاص بالخيار الذي تريده.  
- أو **سحب ملف OSD** وإفلاته مباشرة على **osd_tool.exe**.
